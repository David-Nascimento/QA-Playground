Tabela com Paginação Dinâmica

Pratique testes em tabelas com paginação dinâmica. Esta página é ideal para testar automação de navegação entre páginas, validação de dados em diferentes páginas e controle de paginação em tabelas extensas.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar navegação entre páginas de uma tabela paginada
  • Validar que os dados corretos são exibidos em cada página
  • Testar botões de navegação (Anterior, Próxima, Primeira, Última)
  • Validar informações de paginação (página atual, total de páginas, total de itens)
  • Praticar automação de espera por elementos que aparecem após mudança de página
  • Testar validação de dados em diferentes páginas da tabela
  • Validar comportamento quando não há dados ou apenas uma página

Como Utilizar:

  1. Observe a tabela com dados paginados (10 itens por página)
  2. Use os botões "Anterior" e "Próxima" para navegar entre páginas
  3. Valide que os dados mudam corretamente ao navegar
  4. Verifique as informações de paginação (página atual e total)
  5. Teste os limites (primeira e última página)
  6. Valide que os botões são desabilitados quando apropriado
  7. Use os data-testid para criar testes automatizados

Informações para Testes

Cenários de Teste:

  • Navegar para a próxima página e validar dados
  • Navegar para a página anterior e validar dados
  • Validar que os botões são desabilitados nas extremidades
  • Verificar informações de paginação (página atual, total)
  • Validar contagem total de itens
  • Testar navegação rápida entre múltiplas páginas
  • Validar que os dados mudam corretamente ao navegar

Elementos identificáveis:

  • data-testid="tabela-paginada" - Tabela principal
  • data-testid="tbody-paginada" - Corpo da tabela
  • data-testid="btn-pagina-anterior" - Botão página anterior
  • data-testid="btn-pagina-proxima" - Botão página próxima
  • data-testid="info-paginacao" - Informações de paginação
  • data-testid="total-itens" - Total de itens
  • data-testid="linha-pag-[id]" - Cada linha da tabela

Exemplos de Teste Automatizado:


// Cypress
cy.get('[data-testid="tabela-paginada"]').should('be.visible')
cy.get('[data-testid="tbody-paginada"] tr').should('have.length', 10)
cy.get('[data-testid="btn-pagina-proxima"]').click()
cy.get('[data-testid="info-paginacao"]').should('contain', 'Página 2')
cy.get('[data-testid="tbody-paginada"] tr').should('have.length', 10)
cy.get('[data-testid="btn-pagina-anterior"]').click()
cy.get('[data-testid="info-paginacao"]').should('contain', 'Página 1')
                

// Playwright
await expect(page.getByTestId('tabela-paginada')).toBeVisible()
await expect(page.locator('[data-testid="tbody-paginada"] tr')).toHaveCount(10)
await page.getByTestId('btn-pagina-proxima').click()
await expect(page.getByTestId('info-paginacao')).toContainText('Página 2')
await expect(page.locator('[data-testid="tbody-paginada"] tr')).toHaveCount(10)
await page.getByTestId('btn-pagina-anterior').click()
await expect(page.getByTestId('info-paginacao')).toContainText('Página 1')
                

// Selenium Java
WebElement tabela = driver.findElement(By.cssSelector("[data-testid='tabela-paginada']"));
assert tabela.isDisplayed();
List<WebElement> linhas = driver.findElements(By.cssSelector("[data-testid='tbody-paginada'] tr"));
assert linhas.size() == 10;
driver.findElement(By.cssSelector("[data-testid='btn-pagina-proxima']")).click();
WebElement info = driver.findElement(By.cssSelector("[data-testid='info-paginacao']"));
assert info.getText().contains("Página 2");
                

# Capybara (Ruby)
expect(page).to have_selector('[data-testid="tabela-paginada"]')
expect(page).to have_selector('[data-testid="tbody-paginada"] tr', count: 10)
find('[data-testid="btn-pagina-proxima"]').click
expect(find('[data-testid="info-paginacao"]')).to have_content('Página 2')
expect(page).to have_selector('[data-testid="tbody-paginada"] tr', count: 10)
find('[data-testid="btn-pagina-anterior"]').click
expect(find('[data-testid="info-paginacao"]')).to have_content('Página 1')
