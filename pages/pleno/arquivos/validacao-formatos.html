Validação de Formatos

Pratique testes de validação de formatos de arquivo. Esta página é ideal para testar automação de validação de tipos de arquivo, mensagens de erro para formatos inválidos e limites de tamanho.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar validação de formatos aceitos
  • Validar mensagens de erro para formatos inválidos
  • Praticar automação de validação de tipos de arquivo
  • Testar limites de tamanho de arquivo
  • Validar diferentes tipos de arquivo (imagens, documentos, etc.)
  • Testar validação de extensões de arquivo
  • Validar feedback visual de validação

Como Utilizar:

  1. Selecione diferentes tipos de arquivo nos campos abaixo
  2. Observe as mensagens de validação para formatos aceitos e rejeitados
  3. Teste arquivos que excedem o tamanho máximo
  4. Valide que mensagens de erro aparecem corretamente
  5. Use os data-testid para criar testes automatizados

Validação por Tipo

Teste validação de diferentes formatos de arquivo.

Validação de Tamanho

Teste validação de tamanho máximo de arquivo (máximo 5MB).

Tamanho máximo permitido: 5 MB

Informações para Testes

Cenários de Teste:

  • Testar upload de formatos aceitos
  • Validar mensagens de erro para formatos inválidos
  • Testar arquivos que excedem tamanho máximo
  • Validar diferentes tipos de validação (tipo, tamanho)
  • Testar múltiplas validações simultaneamente

Elementos identificáveis:

  • data-testid="file-images" - Campo imagens
  • data-testid="file-documents" - Campo documentos
  • data-testid="file-csv" - Campo CSV
  • data-testid="file-size" - Campo com limite de tamanho
  • data-testid="validation-images" - Mensagem validação imagens
  • data-testid="validation-documents" - Mensagem validação documentos
  • data-testid="validation-size" - Mensagem validação tamanho

Exemplos de Teste Automatizado:


// Cypress - Testar validação de formatos
// Formato válido
cy.get('[data-testid="file-images"]').selectFile('cypress/fixtures/example.jpg')
cy.get('[data-testid="validation-images"]').should('contain', 'válido')

// Formato inválido (simulado)
cy.get('[data-testid="file-images"]').selectFile('cypress/fixtures/example.pdf')
cy.get('[data-testid="validation-images"]').should('contain', 'inválido')

// Tamanho excedido
cy.get('[data-testid="file-size"]').selectFile('cypress/fixtures/large-file.jpg')
cy.get('[data-testid="validation-size"]').should('contain', 'excede')
                

// Playwright - Testar validação de formatos
// Formato válido
await page.getByTestId('file-images').setInputFiles('tests/fixtures/example.jpg')
await expect(page.getByTestId('validation-images')).toContainText('válido')

// Formato inválido
await page.getByTestId('file-images').setInputFiles('tests/fixtures/example.pdf')
await expect(page.getByTestId('validation-images')).toContainText('inválido')

// Tamanho excedido
await page.getByTestId('file-size').setInputFiles('tests/fixtures/large-file.jpg')
await expect(page.getByTestId('validation-size')).toContainText('excede')
                

// Selenium Java - Testar validação de formatos
WebElement fileInput = driver.findElement(By.cssSelector("[data-testid='file-images']"));
String validFile = System.getProperty("user.dir") + "/src/test/resources/example.jpg";
fileInput.sendKeys(validFile);
Thread.sleep(500);
WebElement validation = driver.findElement(By.cssSelector("[data-testid='validation-images']"));
assert validation.getText().contains("válido");

// Testar formato inválido
fileInput.clear();
String invalidFile = System.getProperty("user.dir") + "/src/test/resources/example.pdf";
fileInput.sendKeys(invalidFile);
Thread.sleep(500);
assert validation.getText().contains("inválido");
                

# Capybara (Ruby) - Testar validação de formatos
# Formato válido
attach_file('file-images', Rails.root.join('spec/fixtures/example.jpg'))
expect(find('[data-testid="validation-images"]')).to have_content('válido')

# Formato inválido
attach_file('file-images', Rails.root.join('spec/fixtures/example.pdf'))
expect(find('[data-testid="validation-images"]')).to have_content('inválido')

# Tamanho excedido
attach_file('file-size', Rails.root.join('spec/fixtures/large-file.jpg'))
expect(find('[data-testid="validation-size"]')).to have_content('excede')
