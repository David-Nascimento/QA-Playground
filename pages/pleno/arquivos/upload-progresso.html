Upload com Progresso

Pratique testes de upload com barra de progresso. Esta página é ideal para testar automação de uploads com feedback visual, validação de progresso e simulação de uploads longos.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar upload com barra de progresso
  • Validar atualização de progresso em tempo real
  • Praticar automação de espera por progresso
  • Testar cancelamento de upload
  • Validar estados de upload (iniciando, em progresso, concluído, erro)
  • Testar uploads de diferentes tamanhos
  • Validar mensagens de status durante o upload

Como Utilizar:

  1. Selecione um arquivo para upload
  2. Clique em "Iniciar Upload" para começar o processo
  3. Observe a barra de progresso sendo atualizada
  4. Veja as mensagens de status mudando durante o upload
  5. Teste o botão "Cancelar" durante o upload
  6. Use os data-testid para criar testes automatizados

Informações para Testes

Cenários de Teste:

  • Iniciar upload e validar progresso
  • Validar atualização da barra de progresso
  • Testar cancelamento de upload
  • Validar mensagens de status
  • Testar upload completo e validação de conclusão
  • Validar informações de arquivo durante upload

Elementos identificáveis:

  • data-testid="file-input-progress" - Campo de seleção
  • data-testid="btn-start-upload" - Botão iniciar upload
  • data-testid="btn-cancel-upload" - Botão cancelar
  • data-testid="upload-status" - Área de status
  • data-testid="progress-bar" - Barra de progresso
  • data-testid="progress-text" - Texto de progresso
  • data-testid="status-text" - Mensagem de status

Exemplos de Teste Automatizado:


// Cypress - Testar upload com progresso
cy.get('[data-testid="file-input-progress"]').selectFile('cypress/fixtures/example.jpg')
cy.get('[data-testid="btn-start-upload"]').should('not.be.disabled').click()
cy.get('[data-testid="upload-status"]').should('be.visible')
cy.get('[data-testid="status-text"]').should('contain', 'Enviando')
cy.get('[data-testid="progress-bar"]', { timeout: 10000 }).should('have.attr', 'style').and('contain', '100%')
cy.get('[data-testid="status-text"]').should('contain', 'Concluído')
                

// Playwright - Testar upload com progresso
await page.getByTestId('file-input-progress').setInputFiles('tests/fixtures/example.jpg')
await expect(page.getByTestId('btn-start-upload')).toBeEnabled()
await page.getByTestId('btn-start-upload').click()
await expect(page.getByTestId('upload-status')).toBeVisible()
await expect(page.getByTestId('status-text')).toContainText('Enviando')
await expect(page.getByTestId('progress-bar')).toHaveAttribute('style', /100%/, { timeout: 10000 })
await expect(page.getByTestId('status-text')).toContainText('Concluído')
                

// Selenium Java - Testar upload com progresso
WebElement fileInput = driver.findElement(By.cssSelector("[data-testid='file-input-progress']"));
fileInput.sendKeys(System.getProperty("user.dir") + "/src/test/resources/example.jpg");
WebElement startBtn = driver.findElement(By.cssSelector("[data-testid='btn-start-upload']"));
assert startBtn.isEnabled();
startBtn.click();
WebElement status = driver.findElement(By.cssSelector("[data-testid='upload-status']"));
assert status.isDisplayed();
WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
wait.until(ExpectedConditions.textToBePresentInElement(
  driver.findElement(By.cssSelector("[data-testid='status-text']")), "Concluído"
));
                

# Capybara (Ruby) - Testar upload com progresso
attach_file('file-input-progress', Rails.root.join('spec/fixtures/example.jpg'))

expect(find('[data-testid="status-text"]')).to have_content('Concluído')
