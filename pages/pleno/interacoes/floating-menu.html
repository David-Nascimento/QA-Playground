Floating Menu

Pratique testes de menus flutuantes e fixos na tela. Esta página é ideal para testar automação de elementos com posição fixa, validação de menus que permanecem visíveis ao rolar a página e interações com elementos flutuantes.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar menus flutuantes com posição fixa
  • Validar que menus permanecem visíveis ao rolar a página
  • Praticar automação de elementos com position: fixed
  • Testar interações com menus flutuantes
  • Validar comportamento em diferentes resoluções de tela
  • Testar menus que aparecem/desaparecem dinamicamente
  • Validar z-index e sobreposição de elementos

Como Utilizar:

  1. Observe o menu flutuante no canto superior direito
  2. Role a página para verificar que o menu permanece fixo
  3. Clique nos itens do menu para testar interações
  4. Valide que o menu está sempre visível
  5. Use os data-testid para criar testes automatizados

Informações para Testes

Cenários de Teste:

  • Validar que menu está visível e fixo
  • Rolar página e validar que menu permanece fixo
  • Clicar em itens do menu e validar ações
  • Testar menu em diferentes resoluções de tela
  • Validar posição do menu (top, right)

Elementos identificáveis:

  • data-testid="floating-menu" - Menu flutuante
  • data-testid="floating-menu-header" - Cabeçalho do menu
  • data-testid="menu-item-home" - Item Home
  • data-testid="menu-item-about" - Item Sobre
  • data-testid="menu-item-contact" - Item Contato
  • data-testid="menu-result" - Área de resultado

Exemplos de Teste Automatizado:


// Cypress - Testar menu flutuante
cy.get('[data-testid="floating-menu"]').should('be.visible')
cy.scrollTo('bottom')
cy.get('[data-testid="floating-menu"]').should('be.visible')
cy.get('[data-testid="menu-item-home"]').click()
cy.get('[data-testid="menu-result"]').should('contain', 'Home')
cy.get('[data-testid="menu-item-about"]').click()
cy.get('[data-testid="menu-result"]').should('contain', 'Sobre')
                

// Playwright - Testar menu flutuante
await expect(page.getByTestId('floating-menu')).toBeVisible()
await page.evaluate(() => window.scrollTo(0, document.body.scrollHeight))
await expect(page.getByTestId('floating-menu')).toBeVisible()
await page.getByTestId('menu-item-home').click()
await expect(page.getByTestId('menu-result')).toContainText('Home')
await page.getByTestId('menu-item-about').click()
await expect(page.getByTestId('menu-result')).toContainText('Sobre')
                

// Selenium Java - Testar menu flutuante
WebElement menu = driver.findElement(By.cssSelector("[data-testid='floating-menu']"));
assert menu.isDisplayed();
((JavascriptExecutor) driver).executeScript("window.scrollTo(0, document.body.scrollHeight)");
Thread.sleep(500);
assert menu.isDisplayed();
driver.findElement(By.cssSelector("[data-testid='menu-item-home']")).click();
WebElement result = driver.findElement(By.cssSelector("[data-testid='menu-result']"));
assert result.getText().contains("Home");
                

# Capybara (Ruby) - Testar menu flutuante
expect(find('[data-testid="floating-menu"]')).to be_visible
page.execute_script('window.scrollTo(0, document.body.scrollHeight)')
expect(find('[data-testid="floating-menu"]')).to be_visible
find('[data-testid="menu-item-home"]').click
expect(find('[data-testid="menu-result"]')).to have_content('Home')
find('[data-testid="menu-item-about"]').click
expect(find('[data-testid="menu-result"]')).to have_content('Sobre')
