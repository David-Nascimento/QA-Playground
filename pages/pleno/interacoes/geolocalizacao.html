Geolocalização

Pratique testes de recursos de localização do navegador. Esta página é ideal para testar automação de geolocalização, validação de permissões de localização e mock de coordenadas em testes automatizados.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar solicitação de permissão de geolocalização
  • Validar captura de coordenadas (latitude e longitude)
  • Praticar mock de geolocalização em testes automatizados
  • Testar tratamento de permissão negada
  • Validar tratamento de erro quando geolocalização não está disponível
  • Testar diferentes coordenadas em diferentes ambientes
  • Validar precisão e formato das coordenadas

Como Utilizar:

  1. Clique no botão "Obter Localização" abaixo
  2. Permita o acesso à localização quando solicitado
  3. Observe as coordenadas exibidas (latitude e longitude)
  4. Teste negando a permissão para ver o tratamento de erro
  5. Use os data-testid para criar testes automatizados

Nota: Em testes automatizados, você precisará mockar a API de geolocalização, pois os navegadores podem não permitir acesso real à localização em ambientes de teste.

Informações para Testes

Cenários de Teste:

  • Solicitar permissão de geolocalização
  • Validar coordenadas capturadas (latitude, longitude)
  • Mockar geolocalização em testes automatizados
  • Testar tratamento de permissão negada
  • Validar tratamento de erro quando não disponível

Elementos identificáveis:

  • data-testid="geo-btn" - Botão obter localização
  • data-testid="geo-result" - Área de resultado

Exemplos de Teste Automatizado:


// Cypress - Mockar geolocalização
cy.window().then((win) => {
  cy.stub(win.navigator.geolocation, 'getCurrentPosition').callsFake((callback) => {
    callback({
      coords: {
        latitude: -23.5505,
        longitude: -46.6333
      }
    });
  });
});

cy.get('[data-testid="geo-btn"]').click()
cy.get('[data-testid="geo-result"]').should('contain', 'Latitude')
cy.get('[data-testid="geo-result"]').should('contain', '-23.5505')
                

// Playwright - Mockar geolocalização
await page.context().grantPermissions(['geolocation'], {
  origin: 'http://localhost:3000'
});

await page.context().setGeolocation({
  latitude: -23.5505,
  longitude: -46.6333
});

await page.getByTestId('geo-btn').click()
await expect(page.getByTestId('geo-result')).toContainText('Latitude')
await expect(page.getByTestId('geo-result')).toContainText('-23.5505')
                

// Selenium Java - Mockar geolocalização (Chrome)
ChromeOptions options = new ChromeOptions();
Map<String, Object> prefs = new HashMap<>();
prefs.put("profile.default_content_setting_values.geolocation", 1);
options.setExperimentalOption("prefs", prefs);

// Definir localização mockada
Map<String, Object> geoLocation = new HashMap<>();
geoLocation.put("latitude", -23.5505);
geoLocation.put("longitude", -46.6333);
geoLocation.put("accuracy", 100);
driver.executeCdpCommand("Emulation.setGeolocationOverride", geoLocation);

driver.findElement(By.cssSelector("[data-testid='geo-btn']")).click();
WebElement result = driver.findElement(By.cssSelector("[data-testid='geo-result']"));
assert result.getText().contains("Latitude");
                

# Capybara (Ruby) - Mockar geolocalização
page.execute_script("
  navigator.geolocation.getCurrentPosition = function(success) {
    success({
      coords: {
        latitude: -23.5505,
        longitude: -46.6333
      }
    });
  };
")

find('[data-testid="geo-btn"]').click
expect(find('[data-testid="geo-result"]')).to have_content('Latitude')
expect(find('[data-testid="geo-result"]')).to have_content('-23.5505')
