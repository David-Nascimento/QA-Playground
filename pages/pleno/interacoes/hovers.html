Hovers

Pratique testes de interações com hover (passar o mouse). Esta página é ideal para testar automação de eventos de mouse, validação de elementos que aparecem ao passar o mouse e menus dropdown que dependem de hover.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar eventos de hover (mouseenter/mouseleave)
  • Validar elementos que aparecem ao passar o mouse
  • Praticar automação de hover em diferentes frameworks
  • Testar menus dropdown que dependem de hover
  • Validar tooltips e informações que aparecem no hover
  • Testar múltiplos elementos com hover na mesma página
  • Validar feedback visual de hover (mudança de cor, tamanho, etc.)

Como Utilizar:

  1. Passe o mouse sobre os elementos abaixo
  2. Observe as informações que aparecem ao fazer hover
  3. Valide que os elementos mudam de aparência ao passar o mouse
  4. Teste com diferentes elementos na página
  5. Use os data-testid para criar testes automatizados

Informações para Testes

Cenários de Teste:

  • Fazer hover sobre elementos e validar informações que aparecem
  • Validar que elementos mudam de aparência ao passar o mouse
  • Testar hover em múltiplos elementos
  • Validar que informações desaparecem ao remover o mouse
  • Testar hover em diferentes resoluções de tela

Elementos identificáveis:

  • data-testid="hover-box-1" - Primeira caixa de hover
  • data-testid="hover-box-2" - Segunda caixa de hover
  • data-testid="hover-box-3" - Terceira caixa de hover
  • data-testid="hover-info-1" - Informação da primeira caixa
  • data-testid="hover-button" - Botão com hover
  • data-testid="hover-tooltip" - Tooltip do botão

Exemplos de Teste Automatizado:


// Cypress - Testar hover
cy.get('[data-testid="hover-box-1"]').trigger('mouseover')
cy.get('[data-testid="hover-info-1"]').should('be.visible')
cy.get('[data-testid="hover-info-1"]').should('contain', 'Informação ao passar o mouse')
cy.get('[data-testid="hover-box-1"]').trigger('mouseout')
cy.get('[data-testid="hover-info-1"]').should('not.be.visible')

// Testar botão com hover
cy.get('[data-testid="hover-button"]').trigger('mouseover')
cy.get('[data-testid="hover-tooltip"]').should('be.visible')
                

// Playwright - Testar hover
await page.getByTestId('hover-box-1').hover()
await expect(page.getByTestId('hover-info-1')).toBeVisible()
await expect(page.getByTestId('hover-info-1')).toContainText('Informação ao passar o mouse')
await page.getByTestId('hover-box-1').hover({ force: false })
await page.mouse.move(0, 0)
await expect(page.getByTestId('hover-info-1')).toBeHidden()

// Testar botão com hover
await page.getByTestId('hover-button').hover()
await expect(page.getByTestId('hover-tooltip')).toBeVisible()
                

// Selenium Java - Testar hover
WebElement hoverBox = driver.findElement(By.cssSelector("[data-testid='hover-box-1']"));
Actions actions = new Actions(driver);
actions.moveToElement(hoverBox).perform();
WebElement hoverInfo = driver.findElement(By.cssSelector("[data-testid='hover-info-1']"));
assert hoverInfo.isDisplayed();
assert hoverInfo.getText().contains("Informação ao passar o mouse");

// Testar botão com hover
WebElement hoverButton = driver.findElement(By.cssSelector("[data-testid='hover-button']"));
actions.moveToElement(hoverButton).perform();
WebElement tooltip = driver.findElement(By.cssSelector("[data-testid='hover-tooltip']"));
assert tooltip.isDisplayed();
                

# Capybara (Ruby) - Testar hover
find('[data-testid="hover-box-1"]').hover
expect(find('[data-testid="hover-info-1"]')).to be_visible
expect(find('[data-testid="hover-info-1"]')).to have_content('Informação ao passar o mouse')

# Testar botão com hover
find('[data-testid="hover-button"]').hover
expect(find('[data-testid="hover-tooltip"]')).to be_visible
