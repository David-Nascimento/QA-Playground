Crie sua Conta

Pratique automação e validação de registro de novos usuários. Todos os elementos possuem id e data-testid para facilitar testes automatizados.

Dados Pessoais

Mínimo 3 caracteres
Digite um email válido
Formato: 000.000.000-00
Você deve ter pelo menos 18 anos
Formato: (00) 00000-0000

Endereço

Formato: 00000-000
Opcional

Segurança

Mínimo 8 caracteres, incluindo letras e números
As senhas devem ser iguais

Informações para Testes

Dados de Teste:

  • Nome: Qualquer nome com mais de 3 caracteres
  • Email: Formato válido (ex: teste@exemplo.com)
  • CPF: Formato 000.000.000-00
  • Data: Deve ter pelo menos 18 anos
  • Senha: Mínimo 8 caracteres com letras e números

Elementos identificáveis:

  • data-testid="input-nome"
  • data-testid="input-email"
  • data-testid="input-cpf"
  • data-testid="input-senha"
  • data-testid="input-confirmar-senha"
  • data-testid="checkbox-termos"
  • data-testid="btn-registrar"

Exemplos de Teste Automatizado:


// Cypress
cy.get('[data-testid="input-nome"]').type('João Silva')
cy.get('[data-testid="input-email"]').type('joao@exemplo.com')
cy.get('[data-testid="input-cpf"]').type('123.456.789-00')
cy.get('[data-testid="input-senha"]').type('Senha123')
cy.get('[data-testid="input-confirmar-senha"]').type('Senha123')
cy.get('[data-testid="checkbox-termos"]').check()
cy.get('[data-testid="btn-registrar"]').click()
                

// Playwright
await page.getByTestId('input-nome').fill('João Silva')
await page.getByTestId('input-email').fill('joao@exemplo.com')
await page.getByTestId('input-cpf').fill('123.456.789-00')
await page.getByTestId('input-senha').fill('Senha123')
await page.getByTestId('input-confirmar-senha').fill('Senha123')
await page.getByTestId('checkbox-termos').check()
await page.getByTestId('btn-registrar').click()
                

// Selenium Java
driver.findElement(By.cssSelector("[data-testid='input-nome']")).sendKeys("João Silva");
driver.findElement(By.cssSelector("[data-testid='input-email']")).sendKeys("joao@exemplo.com");
driver.findElement(By.cssSelector("[data-testid='input-cpf']")).sendKeys("123.456.789-00");
driver.findElement(By.cssSelector("[data-testid='input-senha']")).sendKeys("Senha123");
driver.findElement(By.cssSelector("[data-testid='input-confirmar-senha']")).sendKeys("Senha123");
driver.findElement(By.cssSelector("[data-testid='checkbox-termos']")).click();
driver.findElement(By.cssSelector("[data-testid='btn-registrar']")).click();
