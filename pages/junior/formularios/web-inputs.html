Entradas de Formulário (Web Inputs)

Pratique com diferentes tipos de campos de entrada HTML. Esta página é ideal para testar automação e validação de todos os tipos de inputs disponíveis no HTML5, incluindo texto, email, senha, número, data, hora, cor, range e checkbox.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar preenchimento de diferentes tipos de campos em formulários web
  • Validar comportamento de inputs HTML5 nativos (text, email, password, number, date, time, color, range)
  • Praticar automação de preenchimento de formulários com Selenium, Cypress ou Playwright
  • Validar valores mínimos e máximos em campos numéricos
  • Testar seletores de data e hora nativos do navegador
  • Validar seleção de cores e valores de range (slider)
  • Testar interação com checkboxes e validação de estados

Como Utilizar:

  1. Preencha os diferentes tipos de campos disponíveis no formulário
  2. Teste validações nativas (ex: email deve ter formato válido, número deve estar entre min e max)
  3. Interaja com o seletor de cor e o slider de range
  4. Marque/desmarque o checkbox de termos
  5. Clique em "Enviar Formulário" para ver os dados coletados
  6. Use os data-testid para criar testes automatizados

Informações para Testes

Elementos identificáveis:

  • data-testid="form-web-inputs" - Formulário principal
  • data-testid="input-text" - Campo de texto
  • data-testid="input-email" - Campo de email
  • data-testid="input-password" - Campo de senha
  • data-testid="input-number" - Campo numérico (min: 0, max: 1000)
  • data-testid="input-date" - Seletor de data
  • data-testid="input-time" - Seletor de hora
  • data-testid="input-color" - Seletor de cor
  • data-testid="input-range" - Slider de range (0-100)
  • data-testid="input-checkbox" - Checkbox de termos
  • data-testid="btn-enviar" - Botão de envio
  • data-testid="resultado-formulario" - Área de resultado

Exemplos de Teste Automatizado: 


              // Cypress
              cy.get('[data-testid="input-email"]').type('teste@exemplo.com')
              cy.get('[data-testid="input-password"]').type('senha123')
              cy.get('[data-testid="input-number"]').type('50')
              cy.get('[data-testid="input-checkbox"]').check()
              cy.get('[data-testid="btn-enviar"]').click()
              cy.get('[data-testid="resultado-formulario"]').should('be.visible')
                

              // Playwright
              await page.getByTestId('input-email').fill('teste@exemplo.com')
              await page.getByTestId('input-password').fill('senha123')
              await page.getByTestId('input-number').fill('50')
              await page.getByTestId('input-checkbox').check()
              await page.getByTestId('btn-enviar').click()
              await expect(page.getByTestId('resultado-formulario')).toBeVisible()
                

              // Selenium Java
              driver.findElement(By.cssSelector("[data-testid='input-email']")).sendKeys("teste@exemplo.com");
              driver.findElement(By.cssSelector("[data-testid='input-password']")).sendKeys("senha123");
              driver.findElement(By.cssSelector("[data-testid='input-number']")).sendKeys("50");
              driver.findElement(By.cssSelector("[data-testid='input-checkbox']")).click();
              driver.findElement(By.cssSelector("[data-testid='btn-enviar']")).click();
                

                # Capybara (Ruby)
                visit '/web-inputs.html'
                
                fill_in 'input-email', with: 'teste@exemplo.com'
                fill_in 'input-password', with: 'senha123'
                fill_in 'input-number', with: '50'
                
                find('[data-testid="input-checkbox"]').click
                find('[data-testid="btn-enviar"]').click
                
                expect(page).to have_selector('[data-testid="resultado-formulario"]')
