Recuperação de Senha

Pratique o fluxo completo de recuperação de senha: solicitação, código de verificação e redefinição. Todos os elementos possuem id e data-testid para facilitar testes automatizados.

Etapa 1: Solicitar Recuperação

Digite o e-mail cadastrado para receber o código de recuperação.

Digite o e-mail usado no cadastro

Etapa 2: Verificar Código

Enviamos um código de 6 dígitos para . Digite o código abaixo.

Código de 6 dígitos enviado por e-mail

Etapa 3: Definir Nova Senha

Crie uma nova senha segura para sua conta.

Mínimo 8 caracteres, incluindo letras e números

Informações para Testes

Fluxo de Teste:

  1. Digite um e-mail válido (ex: teste@exemplo.com)
  2. Clique em "Enviar Código de Recuperação"
  3. Digite o código: 123456 (código de teste)
  4. Defina uma nova senha (mínimo 8 caracteres com letras e números)
  5. Confirme a nova senha

Elementos identificáveis:

  • data-testid="etapa-solicitar"
  • data-testid="input-email-recuperacao"
  • data-testid="btn-enviar-codigo"
  • data-testid="etapa-codigo"
  • data-testid="input-codigo-verificacao"
  • data-testid="btn-verificar-codigo"
  • data-testid="etapa-nova-senha"
  • data-testid="input-nova-senha"
  • data-testid="btn-redefinir-senha"
  • data-testid="etapa-sucesso"

Código de Teste:

Use o código 123456 para testes automatizados.

Exemplos de Teste Automatizado:


// Cypress
cy.get('[data-testid="input-email-recuperacao"]').type('teste@exemplo.com')
cy.get('[data-testid="btn-enviar-codigo"]').click()
cy.get('[data-testid="input-codigo-verificacao"]').type('123456')
cy.get('[data-testid="btn-verificar-codigo"]').click()
cy.get('[data-testid="input-nova-senha"]').type('NovaSenha123')
cy.get('[data-testid="input-confirmar-nova-senha"]').type('NovaSenha123')
cy.get('[data-testid="btn-redefinir-senha"]').click()
                

// Playwright
await page.getByTestId('input-email-recuperacao').fill('teste@exemplo.com')
await page.getByTestId('btn-enviar-codigo').click()
