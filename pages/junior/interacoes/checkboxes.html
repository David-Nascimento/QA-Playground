Checkboxes

Pratique a seleção múltipla de opções com checkboxes. Esta página é ideal para testar seleção múltipla, validação de estados checked/unchecked e automação de interações com elementos checkbox HTML. Múltiplas opções podem ser selecionadas simultaneamente.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar seleção múltipla de checkboxes independentes
  • Validar estados checked e unchecked de cada checkbox
  • Praticar automação de marcar/desmarcar checkboxes
  • Testar seleção/deseleção em massa (selecionar todos, desmarcar todos)
  • Validar checkboxes obrigatórios (ex: termos e condições)
  • Testar grupos de checkboxes relacionados (ex: habilidades, interesses)
  • Validar feedback visual e contagem de itens selecionados

Como Utilizar:

  1. Marque e desmarque checkboxes individuais
  2. Valide que múltiplos checkboxes podem estar selecionados simultaneamente
  3. Teste os botões "Selecionar Todos" e "Desmarcar Todos"
  4. Valide o feedback visual e as mensagens de seleção
  5. Interaja com diferentes grupos de checkboxes (Termos, Habilidades, Interesses)
  6. Use os data-testid para criar testes automatizados

Informações para Testes

Cenários de Teste:

  • Marcar um checkbox individual
  • Desmarcar um checkbox
  • Validar que múltiplos checkboxes podem estar selecionados
  • Verificar estado checked/unchecked de cada checkbox
  • Testar seleção/deseleção em massa
  • Validar valores e labels de cada checkbox

Elementos identificáveis:

  • data-testid="checkbox-termos", data-testid="checkbox-privacidade", etc.
  • data-testid="checkbox-selenium", data-testid="checkbox-cypress", etc.
  • data-testid="btn-selecionar-todos"
  • data-testid="btn-desmarcar-todos"

Exemplos de Teste Automatizado:


// Cypress
cy.get('[data-testid="checkbox-termos"]').check()
cy.get('[data-testid="checkbox-selenium"]').check()
cy.get('[data-testid="checkbox-cypress"]').check()
cy.get('[data-testid="checkbox-termos"]').should('be.checked')
cy.get('[data-testid="btn-selecionar-todos"]').click()
                

// Playwright
await page.getByTestId('checkbox-termos').check()
await page.getByTestId('checkbox-selenium').check()
await page.getByTestId('checkbox-cypress').check()
await expect(page.getByTestId('checkbox-termos')).toBeChecked()
await page.getByTestId('btn-selecionar-todos').click()
                

// Selenium Java
driver.findElement(By.cssSelector("[data-testid='checkbox-termos']")).click();
driver.findElement(By.cssSelector("[data-testid='checkbox-selenium']")).click();
driver.findElement(By.cssSelector("[data-testid='checkbox-cypress']")).click();
driver.findElement(By.cssSelector("[data-testid='btn-selecionar-todos']")).click();
                

# Capybara (Ruby)
check 'termos'
find('[data-testid="checkbox-selenium"]').click
find('[data-testid="checkbox-cypress"]').click
find('[data-testid="btn-selecionar-todos"]').click
