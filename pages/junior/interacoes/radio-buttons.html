Botões de Rádio

Pratique a seleção de opções exclusivas com radio buttons. Esta página é ideal para testar seleção única, validação de grupos de opções e automação de interações com elementos radio HTML. Apenas uma opção pode ser selecionada por vez em cada grupo.

Sobre esta Seção

Atividades do Dia a Dia:

  • Testar seleção de opções exclusivas em grupos de radio buttons
  • Validar que apenas uma opção pode estar selecionada por grupo
  • Praticar automação de seleção por valor, label ou data-testid
  • Testar mudança de seleção entre opções do mesmo grupo
  • Validar estado inicial (nenhuma opção selecionada ou opção padrão)
  • Testar múltiplos grupos de radio buttons na mesma página
  • Validar feedback visual e eventos de mudança

Como Utilizar:

  1. Clique em uma opção de radio button em qualquer grupo
  2. Valide que apenas uma opção está selecionada por grupo
  3. Teste selecionar outra opção no mesmo grupo e verifique que a anterior é desmarcada
  4. Interaja com diferentes grupos de radio buttons simultaneamente
  5. Valide o feedback visual e as mensagens de seleção
  6. Use os data-testid para criar testes automatizados

Informações para Testes

Cenários de Teste:

  • Selecionar uma opção de rádio
  • Validar que apenas uma opção pode estar selecionada por grupo
  • Verificar estado selecionado de cada opção
  • Testar mudança de seleção entre opções
  • Validar valores e labels de cada opção

Elementos identificáveis:

  • data-testid="radiogroup-genero"
  • data-testid="radio-masculino", data-testid="radio-feminino", etc.
  • data-testid="radiogroup-experiencia"
  • data-testid="radiogroup-preferencia"

Exemplos de Teste Automatizado:


// Cypress
cy.get('[data-testid="radio-masculino"]').check()
cy.get('[data-testid="radio-masculino"]').should('be.checked')
cy.get('[data-testid="radio-junior"]').check()
cy.get('[data-testid="radio-automatizado"]').check()
                

// Playwright
await page.getByTestId('radio-masculino').check()
await expect(page.getByTestId('radio-masculino')).toBeChecked()
await page.getByTestId('radio-junior').check()
await page.getByTestId('radio-automatizado').check()
                

// Selenium Java
driver.findElement(By.cssSelector("[data-testid='radio-masculino']")).click();
driver.findElement(By.cssSelector("[data-testid='radio-junior']")).click();
driver.findElement(By.cssSelector("[data-testid='radio-automatizado']")).click();
                

# Capybara (Ruby)
choose 'radio-masculino'
find('[data-testid="radio-junior"]').click
find('[data-testid="radio-automatizado"]').click
